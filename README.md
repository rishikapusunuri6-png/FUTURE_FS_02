# 📊 Mini CRM Dashboard

A simple and responsive **Client Lead Management System (Mini CRM)** built using **HTML, CSS, and JavaScript**.

---

## 🚀 Features

* 🔐 Admin Login (Username & Password)
* ➕ Add New Leads (Name, Email, Status, Notes)
* 📋 View Leads in Table Format
* ✏️ Edit Leads
* 🗑️ Delete Leads
* 🔍 Search Leads (by Name / Email)
* 📊 Dashboard Analytics:

  * Total Leads
  * New Leads
  * Contacted Leads
  * Converted Leads
* 📝 Notes for each lead
* 💾 Data stored in **LocalStorage**
* 🎨 Modern UI (Cream + Green Theme)
